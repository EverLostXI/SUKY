# Suky

A virtual CD machine for your local music collection.

## Features

* Easy to use
* Simple, focused music experience
* Smooth and natural UI/UX



## Getting Started

1. Visit our project website. 

2. Choose your music library folder, each album under this library should be an individual folder with its title as name, containing every song in this album in mp3/flac format. 

3. Sit back and enjoy. 

4. For a quick jumpstart, left/right arrow or mouse wheel to choose album, left click/enter/space to play an album, esc to exit. You can either hover your cursor on the lower half of the screen to show progress bar, or simply drag the rolling CD to control it.
